# 如何自我介绍，能提升面试通过率
我面试过几千人，总结了一套面试通过率很高的自我介绍。你只要照着做，面试一定会很顺利。

咱们先来回想一下，你去面试的时候是怎么做自我介绍的？是不是一上来就说：面试官你好，我叫什么名字，毕业于哪个学校哪个专业。

然后开始按时间顺序讲：在学校当过班长或者是什么职位、参加过什么项目，第一份实习在哪儿干了什么，第二份工作又在哪儿干了什么……

最后再来一句，我吃苦耐劳、责任心强、抗压能力强，我非常希望能加入贵公司。

停停停。如果你现在还在用这种套路，真的会错过很多本该属于你的机会。

你想想，面试官手里明明有你的简历，他真想听你背一遍吗？他想在短短两三分钟里，直接听到——你跟我们需求最匹配的那个结果是什么，而不是你的个人流水账。

# 那到底该怎么做？

我跟你讲一个三段式的方法。而且我会教你在每一段里“抛钩子”——就是留一个引子，让面试官主动追问你。这样一来，整场面试的节奏就被你掌控了。

第一段，先把基本条件讲清楚，但要极简。

你只需要说：面试官您好，我是谁，毕业于哪个学校什么专业。就这一两句话。

这里有两个小技巧。

一个是扬长避短：如果本科学校一般，你就直接说最高学历；如果专业跟岗位不相关，就补一句“你通过自学或培训补了哪些相关课程”；要是你的GPA很高或者排名靠前，一定得提一下。

另一个技巧是要啥给啥：岗位要求什么，你就给什么。比如说要求有留学经历，你没有，那你可以说你有英语证书、能用英语工作、平时也关注国外行业趋势。跟岗位没关系的事，一个字都别多说。

你这样讲完，面试官就会觉得：嗯，基本条件够了，可以继续听。

然后马上抛第一个钩子。你可以这么说：我有三年新媒体运营经验，其中有一个项目帮公司实现了非常明显的增长。这个项目能做成，其实就靠我接下来要说的三项核心能力。

你看，这个钩子一出去，面试官自然就想听你下面讲什么。

第二段，需要展示你的核心能力，这是整个自我介绍最关键的部分。

记住，不要再说你干过哪些工作，而要讲你具备什么能力。

那怎么知道对方想要什么能力？很简单。你去一个公司面试之前，把这个岗位的招聘要求复制下来，用AI帮你提炼出最核心的三四个关键词。当然，你自己也得过一遍，不能全靠机器。

举个例子，你去面新媒体运营，提炼出来的可能是：内容策划能力、数据分析能力、用户增长能力。

那你就直接看着面试官的眼睛，很自信地说：根据我自己的实践经验，我觉得这个岗位最核心的就是这三项能力。然后马上用真实案例来证明。

比如你可以这样说：在上一家公司，我负责公众号运营。我分析了后台数据，发现用户最喜欢看职场干货，于是就策划了一个“职场进阶”系列文章。结果三个月内，公众号阅读量增长了5倍，涨了八万粉丝，转化率提升了70%。

你看，刚才钩子里说的“非常明显的增长”，就是这组数据。你不是在吹牛，你有实实在在的结果。

讲完这个例子，再抛第二个钩子。你可以说：其实这个项目上线第一周，数据特别差，我们差点就要砍掉了。但后来我们做了一个关键调整……

说到这里你可以停一下，面试官大概率会追问：什么调整？这时候你就从被动回答，变成了主动讲故事。

之前我面试过一位候选人，她面试时逻辑特别清晰，懂得把自己的优势直接展示出来，还总能在关键地方埋钩子。入职后她工作表现也很出色，没多久就晋升了。

当你这样介绍完，面试官看你的眼神都不一样了。因为你能把逻辑理清楚，知道对方要什么，还能拿出证据。这已经超过绝大多数候选人了。

第三段，建立主观的情感连接。

很多人到结尾就习惯拍马屁：说贵公司平台很大，在行业内很有名，我非常想加入。这种话太虚了，十个面试有九个这么说，面试官耳朵都起茧子了。

你要换成主观的连接。比如你可以说：我之前在某家公司实习过，那种快节奏、能逼人成长的环境，我特别适应。我是真心喜欢这个行业，也特别热爱这个岗位。入职后，我一定好好干，拿成绩说话。所以我这次找工作，不是海投，是经过认真考虑，坚定地把贵公司作为首选。

你发现区别没有？客观夸公司那是面子上的客套，而主观讲自己的真实感受才是真正打动人的里子。你是在告诉面试官：我不是来投路无路随便投的，我是带着真实的热爱，专门为你而来的。

心理学上有个现象，人会本能地对坚定选择自己的人产生好感。面试官也是人，当他感受到你的真诚，心里就已经偏向你了。

# 最后总结一下。

这套方法的核心就一句话：每一段都留一个钩子，让面试官追着你问，你就牢牢掌握了面试的主动权。

下次遇到心动的公司，可以试试这个模板。相信你的面试通过率会明显提升。
