## 2. 使用规则

1. 每轮优化后按同一批样本复测，确保可横向对比。
2. 样本原文以 `批量录入.txt` 与 `批量文档` 当前版本为准，禁止评测时临时替换文本。
3. 图谱评估先看“召回命中率 + 关系类型准确率”，再看可视化效果。
4. QA 评估必须看证据回引是否命中正确样本。

## 7. 当前评测关注点（本轮）

1. 主链路召回命中：G1/G2 内部的关系召回是否稳定上升。
2. 误连控制：G4 不应大量误连到 G1。

4. QA 一致性：同一问题在证据充分时不应随机退化为“无证据”。

## 8. 一轮评测问法与验收标准（可直接执行）

### 8.1 图谱评测问法

1. 触发内化后导出图谱 JSON，统计主链路边命中率（6.1）。
2. 检查误连：6.3 中“默认不连”是否被大量连上。
3. 随机抽 10 条边看关系类型是否合理（`related/supports/weak_related`）。

### 8.2 QA 评测问法

1. `Prompt A/B test 的核心五步是什么？`
2. `意图识别准确率评估核心指标有哪些？`
3. `这个问题和刚才 PRD 问题有没有明确关联？没有请明确说没有。`
4. `根据现有笔记，给我一个最小可执行的 AI PRD 模板清单。`
5. `面试表达结构化方法里，为什么说“分点不等于结构化”？`

### 8.3 验收阈值（第一版）

- 主链路边命中率（6.1）：`>= 70%`
- 默认不连误连率（6.3）：`<= 15%`
- QA 证据回引正确率（5问）：`>= 4/5`
- QA“无证据”误拒答率（证据足够却拒答）：`<= 20%`
